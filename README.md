# 高等电动力学课程笔记模板

## 文件说明

### 主要文件
- **`高等电动力学课程总结.tex`** - 完整的LaTeX笔记模板
- **`课程大纲.md`** - 详细的课程大纲（Markdown格式）
- **`README.md`** - 使用说明文档

### 编译输出
- **`高等电动力学课程总结.pdf`** - 编译生成的PDF文档

## 使用方法

### 1. 环境要求
- **TeX发行版**：TeX Live 2020+ 或 MiKTeX
- **编译引擎**：XeLaTeX（推荐）或 LuaLaTeX
- **中文字体**：系统需安装中文字体（Windows通常已内置）

### 2. 编译命令
```bash
# 第一次编译（生成目录和引用）
xelatex 高等电动力学课程总结.tex

# 第二次编译（更新交叉引用）
xelatex 高等电动力学课程总结.tex

# 第三次编译（完善目录）
xelatex 高等电动力学课程总结.tex
```

或者使用自动化工具：
```bash
# 使用latexmk自动编译
latexmk -xelatex 高等电动力学课程总结.tex
```

### 3. 自定义内容

#### 修改个人信息
在文档开头部分修改：
```latex
\title{高等电动力学课程总结笔记}
\author{你的姓名}
\date{\today}
```

#### 添加章节内容
在每个章节的相应位置添加具体内容：
```latex
\section{章节标题}
\subsection{小节标题}
内容...
```

#### 插入数学公式
- 行内公式：`$E = mc^2$`
- 独立公式：
```latex
\begin{equation}
    \nabla \cdot \mathbf{E} = \frac{\rho}{\varepsilon_0}
\end{equation}
```

#### 插入图片
```latex
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.8\textwidth]{图片文件名}
    \caption{图片说明}
    \label{fig:label}
\end{figure}
```

#### 插入代码
```latex
\begin{lstlisting}[language=Python]
# Python代码示例
import numpy as np
x = np.linspace(0, 2*np.pi, 100)
y = np.sin(x)
\end{lstlisting}
```

## 模板特性

### 已包含的LaTeX包
- **中文支持**：`ctex`, `xeCJK`
- **数学公式**：`amsmath`, `amssymb`, `mathtools`
- **图表绘制**：`graphicx`, `tikz`, `pgfplots`
- **页面布局**：`geometry`, `fancyhdr`
- **超链接**：`hyperref`
- **代码高亮**：`listings`
- **表格美化**：`booktabs`, `multirow`

### 定理环境
模板已定义以下定理环境：
- `\begin{theorem}...\end{theorem}` - 定理
- `\begin{lemma}...\end{lemma}` - 引理
- `\begin{definition}...\end{definition}` - 定义
- `\begin{example}...\end{example}` - 例题
- `\begin{exercise}...\end{exercise}` - 练习

### 页面设置
- **纸张大小**：A4
- **字体大小**：12pt
- **页边距**：左右2.5cm，上下3cm
- **页眉页脚**：包含标题、日期和页码

## 课程大纲概览

本课程共13章，涵盖以下内容：

1. **基础知识回顾** - 麦克斯韦方程组、边界条件
2. **静电场** - 泊松方程、分离变量法、镜像法
3. **静电场（续）** - 多极展开、电偶极子
4. **电磁波** - 波动方程、平面波、能量和动量
5. **电磁波的反射和折射** - 斯涅尔定律、菲涅尔公式
6. **波导** - 矩形波导、TE/TM模
7. **圆波导** - 圆柱坐标系、贝塞尔函数
8. **介质加载波导** - 部分填充介质
9. **介质加载圆波导和介质波导** - 介质圆波导、光纤
10. **谐振腔** - 矩形/圆柱谐振腔、品质因数
11. **谐振腔链和空间谐波** - 耦合谐振腔、周期结构
12. **矢势和运动电荷的场** - 电磁势、达朗贝尔方程
13. **运动电荷的场和同步辐射** - 加速电荷辐射、同步辐射

## 学习建议

### 使用模板的方法
1. **按章节学习**：根据课程进度，逐章完善笔记内容
2. **添加例题**：在相应章节添加典型例题和解答
3. **记录推导**：详细记录重要的数学推导过程
4. **总结要点**：每章结束后添加要点总结
5. **练习习题**：结合作业中的习题进行练习

### 笔记技巧
- 使用`\textbf{}`标记重要概念
- 用`\textcolor{red}{}`突出关键公式
- 使用定理环境框出重要结论
- 定期编译检查LaTeX语法错误

## 常见问题

### 编译问题
**Q**: 编译时出现中文字体错误
**A**: 确保使用XeLaTeX编译，系统已安装中文字体

**Q**: 目录不显示
**A**: 需要编译两次以上，第一次生成`.toc`文件，第二次读取

**Q**: 图片不显示
**A**: 检查图片路径是否正确，支持`.pdf`, `.png`, `.jpg`格式

### 内容问题
**Q**: 数学公式显示异常
**A**: 检查数学模式是否正确，确保所有`$`配对

**Q**: 参考文献引用错误
**A**: 运行两次编译，确保`.aux`文件更新
