# 高等电动力学课程大纲

## 课程概述
高等电动力学是物理和电子工程专业的重要基础课程，本课程系统介绍电磁场理论、电磁波传播、导波系统和辐射现象等内容。

## 课程结构（共13章）

### 第一章：基础知识回顾
- **主要内容**：麦克斯韦方程组、边界条件、矢量分析
- **重点**：电磁场基本方程的物理意义和应用
- **课件**：高等电动力学1-基础知识回顾.pdf

### 第二章：静电场
- **主要内容**：泊松方程、拉普拉斯方程、分离变量法
- **重点**：静电场的求解方法
- **课件**：高等电动力学2-静电场1-简.pdf

### 第三章：静电场（续）
- **主要内容**：镜像法、多极展开、电偶极子
- **重点**：复杂边界条件下的静电场问题
- **课件**：高等电动力学3-静电场2.pdf

### 第四章：电磁波
- **主要内容**：波动方程、平面电磁波、电磁波的能量和动量
- **重点**：电磁波的基本特性和传播规律
- **课件**：高等电动力学4-电磁波(1).pdf

### 第五章：电磁波的反射和折射
- **主要内容**：斯涅尔定律、菲涅尔公式、导波系统简介
- **重点**：电磁波在不同介质界面的行为
- **课件**：高等电动力学5-电磁波反射折射-导波系统简介(1).pdf

### 第六章：波导
- **主要内容**：矩形波导、TE模和TM模、截止频率
- **重点**：波导的传输特性和模式分析
- **课件**：高等电动力学6-波导(1).pdf

### 第七章：圆波导
- **主要内容**：圆柱坐标系中的波动方程、贝塞尔函数应用
- **重点**：圆波导的模式和场分布
- **课件**：高等电动力学7-圆波导(1).pdf

### 第八章：介质加载波导
- **主要内容**：部分填充介质的波导、特征方程求解
- **重点**：介质对波导特性的影响
- **课件**：高等电动力学8-介质加载波导(1).pdf

### 第九章：介质加载圆波导和介质波导
- **主要内容**：介质圆波导、光纤原理
- **重点**：介质波导的导模条件和泄漏模
- **课件**：高等电动力学9-介质加载圆波导-介质波导(1).pdf

### 第十章：谐振腔
- **主要内容**：矩形谐振腔、圆柱谐振腔、品质因数
- **重点**：谐振腔的谐振特性和模式分析
- **课件**：高等电动力学10-谐振腔.pdf

### 第十一章：谐振腔链和空间谐波
- **主要内容**：耦合谐振腔、周期结构、布洛赫定理
- **重点**：周期结构的色散关系和带隙
- **课件**：高等电动力学11-谐振腔链-空间谐波(1).pdf

### 第十二章：矢势和运动电荷的场
- **主要内容**：电磁势、洛伦兹规范、达朗贝尔方程、李纳-维谢尔势
- **重点**：运动电荷产生的电磁场
- **课件**：高等电动力学-12-矢势-运动电荷的场.pdf

### 第十三章：运动电荷的场和同步辐射
- **主要内容**：加速电荷的辐射、拉莫尔公式、同步辐射
- **重点**：辐射的特性和应用
- **课件**：高等电动力学13-运动电荷的场-同步辐射(3).pdf

## 重要公式汇总

### 基本方程
- **麦克斯韦方程组**
  - ∇·E = ρ/ε₀
  - ∇·B = 0
  - ∇×E = -∂B/∂t
  - ∇×B = μ₀J + μ₀ε₀∂E/∂t

- **波动方程**
  - ∇²E - με∂²E/∂t² = 0

- **边界条件**
  - E₁t = E₂t
  - B₁n = B₂n

### 重要常数
- 真空介电常数：ε₀ = 8.854×10⁻¹² F/m
- 真空磁导率：μ₀ = 4π×10⁻⁷ H/m
- 光速：c = 3×10⁸ m/s

## 学习建议

### 重点掌握内容
1. 麦克斯韦方程组及其物理意义
2. 电磁波的传播特性和边界条件
3. 各种导波系统的模式分析
4. 运动电荷的辐射特性

### 学习方法
1. **理论推导与物理图像相结合**：既要掌握数学推导，又要理解物理含义
2. **多做习题**：通过练习加深对理论的理解
3. **注意数学技巧**：熟练运用不同坐标系下的数学方法
4. **建立知识框架**：将各章节内容联系起来，形成完整的知识体系

### 参考书目
- J.D. Jackson, *Classical Electrodynamics*, 3rd ed.
- D.J. Griffiths, *Introduction to Electrodynamics*, 4th ed.
- 郭硕鸿, 《电动力学》, 高等教育出版社
